# Sweet Shop Management System

A full-stack Sweet Shop Management System built using the MERN stack.
It supports user registration and login, role-based access (Admin/User),
sweet inventory management, and purchasing functionality.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB (Atlas)
- Authentication: JWT

## Features
- User authentication (Register/Login)
- Admin panel for managing sweets and stock
- Customers can view and purchase sweets
- Role-based access control

## Setup Instructions
1. Clone the repository
2. Install dependencies for both frontend and backend
3. Add `.env` file with MongoDB URI and JWT secret
4. Run backend and frontend servers
